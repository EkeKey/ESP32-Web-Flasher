# EkeKey ESP32-S3 Web Flasher by https://ekekey.com

Static GitHub Pages flasher for EkeKey ESP32-S3 firmware.

## Files

- `index.html`: Web flasher page.
- `manifest.json`: ESP Web Tools manifest.
- `EkeKey-ESP32S3-16MB-full.bin`: full merged firmware at offset `0x0`.

## GitHub Pages

Enable Pages from the repository settings:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

Then open the GitHub Pages URL in Chrome or Edge.
